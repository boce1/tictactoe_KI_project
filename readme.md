## Hot to run
Install ```pygame``` graphics library. Make sure ```pip``` is added to the ```PATH``` variable.
```bash
pip install pygame
```
Run the aplication
```bash
python3 main.py
```

### For windows
There's an extra option for windows machines. To start the game run ```TicTacToe.exe```.
